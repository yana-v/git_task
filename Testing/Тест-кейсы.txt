Тест-кейс № 1.Ввод одинакового города для отправления и прибытия
1. Зайти на сайт https://rzd.poezda.net/.
2. Ввести название города в поле для города отправления  (Бобруйск).
3. Ввести название города в поле для города прибытия (Бобруйск). 
4. В поле с календарём выбрать завтрашнюю дату.
5. Нажать на кнопку “Искать билеты”.
Ожидаемый результат: поле для города прибытия подсветится красным и при наведении на него мышкой появится надпись “Укажите другую станцию назначения”.

Тест-кейс № 4. Заказ билета без ввода личных данных
1. Зайти на сайт https://rzd.poezda.net/.
2. Ввести название города в поле для города отправления  (Бобруйск).
3. Ввести название города в поле для города прибытия (Минск). 
4. В поле с календарём выбрать завтрашнюю дату.
5. Нажать на кнопку “Искать билеты”.
6. Выбрать  первый в списке поезд с наличем мест  и нажать кнопку "Выбрать места".
7. Выбрать первый в списке тип вагона со свободными местами и нажать кнопку “Выбрать места”.
8. В разделе “Количество пассажиров” выбрать тип и количество билетов, нажав нужную цифру в вариантах для типов билета.
9. Выбрать  номер вагона, нажав один из номеров, находящихся слева над схемой вагона .
10. Выбрать на схеме вагона номер свободного места.
11. Нажать кнопку “Дальше”.
12. Оставить все поля с личными данными не заполненными и нажать кнопку “Дальше”
Ожидаемый результат: появится  сообщение об ошибке с указанием, какие поля остались незаполненными.
